Azure - Core Services

Azure 가상머신

  • 실제 컴퓨터의 소프트웨어 emulation
  • 가상 프로세서, 메모리, 스토리지, 네트워크 기능 포함
  • 완벽한 제어와 사용자 지정이 가능한 IaaS 서비스

Azure App Service

  • 웹앱과 API를 빠르게 빌드/배포하고 크기를 조정할 수 있는 완전 관리형 플랫폼
  • .NET, .NET Core, Node.js, Java, Python, PHP, …
  • 엔터프라이즈급 성능/보안/규정 준수 요구사항을 충족하는 PaaS 서비스

Azure Container Service

  • 운영체제를 관리하지 않아도 되며 필요 시 적절하게 변경할 있는 가벼운 가상화 환경
  • Azure Container Instances: 가상 머신이나 추가 서비스를 관리할 필요없이Azure에서 컨테이너를 실행할 수 있는 PaaS 서비스
  • Azure Kubernetes Services: 분산 아키텍처와 대량의 컨테이너를 다루는데 적합한 컨테이너 오케스트레이션 서비스

Azure Networking Services

  • Azure 가상 네트워크: Azure 리소스가 다른 리소스/인터넷/온프레미스 네 트워크와 통신할 수 있게 함
  • VPN gateway: 공용 인터넷을 통해 Azure 가상 네트워크와 온프레미스 위치 간에 암호화된 트래픽 전송
  • Azure Expressroute: 연결 공급자가 제공하는 사설 연결을 통해 온프레미스 네트워크를 Azure로 확장

Azure Storage Services

  • Container Storage (Blob): 텍스트, 이진 데이터 같은 대량의 비구조적 데이터 저장 – object stroage
  • Disk Storage: 가상 머신, 애플리케이션 및 기타 서비스가 액세스하고 사용할 수 있는 디스크 제공
  • Azure Files: 표준 SMB (Server Message Block) 프로토콜을 사용하여 액세스 할 수 있는 고가용성 네트워크 파일 공유 설정

Storage 액세스 계층

Azure Database Services

  • Azure Cosmos Database: 처리량 및 스토리지 크기를 탄력적이고 독립적으로 조정할 수 있는 전역 분산 데이터베이스 서비스 – NoSQL 엔진 포함
  • Azure SQL Database: MS SQL Server 데이터베이스 엔진을 기반으로 하는 서비 스형 관계형 데이터베이스 (DaaS)
  • Azure Database for MySQL: 앱 개발자용 관리형 MySQL 데이터베이스 서비스
  • Azure Database for PostgreSQL: 오픈 소스 Postgres 데이터베이스 엔진 기반 관 계형 데이터베이스 서비스

Azure SQL Managed Instance

  • SQL server 고객이 최소한의 애플리케이션과 데이터베이스 변경으로 lift-and-shift 방식으로 on-premise application을 클라우드로 migration 할 수 있게 함
  • 항상 최신 상태로 유지되는 관리형 PaaS
  • 모든 PaaS 기능 제공 – 자동 패치, 버전 업데이트, 자동 백업, 고가용성 등

Azure Martketplace

  • 다양한 서비스 공급자의 Azure 인증 받은 애플리케이션과 서비스 제공
    • 가상 머신과 데이터베이스 이미지
    • 애플리케이션 빌드/배포용 소프트웨어
    • 개발자 도구
    • 10000개 이상
